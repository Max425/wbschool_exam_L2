Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```

Вывод:
2
1

В функции `test()`, переменная `x` объявлена в сигнатуре функции, и ей присваивается значение 1 до выполнения оператора `defer`. 
Затем, при выполнении отложенной функции, значение `x` увеличивается на 1. Таким образом, к моменту возврата значения `x`, 
оно уже увеличено на 1 благодаря оператору `defer`, поэтому возвращается значение 2.

В функции `anotherTest()`, переменная `x` объявлена внутри функции. Значение `x` устанавливается равным 1, но оператор `defer` 
увеличивает его только после возврата значения. Таким образом, возвращается исходное значение `x`, которое равно 1.

```
